## Install

```console
$ npm i it-length
```

N.b. will consume the iterable

## Usage

```javascript
import length from 'it-length'

// This can also be an iterator, async iterator, generator, etc
const values = [0, 1, 2, 3, 4]

const res = await length(values)

console.info(res) // 5
```
